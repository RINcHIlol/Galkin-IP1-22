Окно авторизации

При запуске приложения, требуется пройти авторизацию, чтобы продолжить работу, если данные корректные, будет открыт экран в зависимости от роли, если некорректные, напишется ошибка.

В данном окне администратор может добавить нового пользователя, если поля окажутся пустые или пользователь с таким логином уже существует, будет написана ошибка.
